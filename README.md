## üöÄ Deployment to Vercel

### Quick Deploy

1. Install Vercel CLI (if not already installed):
   ```bash
   npm install -g vercel
   ```

2. Navigate to the project directory:
   ```bash
   cd "c:\Users\THIS\Documents\BTEHub Land Page"
   ```

3. Deploy to Vercel:
   ```bash
   vercel
   ```

4. Follow the prompts:
   - Set up and deploy? **Y**
   - Which scope? Select your account
   - Link to existing project? **N**
   - Project name? **btehub-solutions** (or your preferred name)
   - Directory? **./** (current directory)
   - Override settings? **N**

### Alternative: Deploy via Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Click "Add New Project"
3. Import your Git repository or drag and drop the project folder
4. Vercel will automatically detect it as a static site
5. Click "Deploy"
